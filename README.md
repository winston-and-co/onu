# ONU

A roguelike deck-builder inspired heavily by Slay the Spire and Balatro

Team:

## Requirements
Unity v2022.3.21f1

## Getting Started
1. Clone the repo
2. In Unity Hub, click "Add" or "Add project from disk"
3. Select the repo
